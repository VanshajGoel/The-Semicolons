Parko

RFID based car parking system
Description - A system to automate the parking service with online payments based on parking duration with RFID cards. Keeping log of entry and exit vehicles for security reasons. Payments can be done through the mobile or web app.

Frameworks - .NET Frameworks

Tools & Softwares - Visual Studio, MySQL, PhpMyAdmin
